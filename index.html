Projects

Description du projet

    Développement d’une application web interactive mettant en lumière 77 femmes ayant marqué l’histoire de Paris, grâce à une visualisation cartographique.

  • Données : exploitation des données Open Data de la Mairie de Paris – Femmes illustres à Paris – pour les portraits et descriptions.
  • Géolocalisation : utilisation de l’API Nominatim (OpenStreetMap) pour obtenir les coordonnées des adresses et les afficher sur une carte Leaflet.
  • Interface utilisateur : navigation via une liste filtrable par catégories (scientifiques, artistes, politiques, etc), avec affichage dynamique du lieu correspondant sur la carte.
  • Technologies utilisées : HTML, CSS, JavaScript.
  • Projet réalisé avec Hanane Benabid et Camille Lebigot.

Description du projet

    Réalisation d’un site web solidaire visant à faciliter et promouvoir l’adoption responsable d’animaux abandonnés.

  • Affichage en grille : une sélection d’animaux est affichée sous forme de grille CSS Grid sur la page d’accueil.
  • Interface utilisateur : navigation via une liste filtrable par catégories et par ville avec affichage sur une deuxième page.
  • Technologies front-end utilisées : HTML, CSS, JavaScript.
  • Conception de base de données back-end : conception et préparation d’une base PostgreSQL hébergée sur Neon, destinée à gérer les candidatures d’adoption (connexion au site non encore active).
  • Projet individuel
